# Virtual-World-Using-Ultrasonic-Sensor-in-Ardiuno
This project demonstrates the virtual world in a graphical way that will track a simple scene in the real world. As part of the project, the virtual world will track both the position and the color of a target in the real world. 

please upload the Ardiuno code to your Ardiuno Board.

after Uploading ,Run the Python Code in VIDLE for Python.
